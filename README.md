# Cadmium
Cadmium is a python command line application made to conveniently download youtube videos for without the risk of accidentally installing malware or getting overwhelmed with ads. It's a great all in one tool that has all the features you'll ever need for downloading youtube videos.

## Features

* ✔️ **Simple to Use**
    * Cadmium has a very simple command line user interface with great tutorials and documentation.
* 🛡️ **Safe**
    * Cadmium's source code as well as it's dependencies are publicly visible and completely open source ensuring full transparency as to how the program functions.
* ✨ **High Quality Downloads**
    * Cadmium can download super high quality video up to 4320p (8K) resolution.  
* 🔄 **Automatic File Conversion**
    * Cadmium converts downloaded videos into highly supported file formats for you automatically while allowing you to also control what kind of files they are converted to as well. 
* ⚙️ **Configurable**
    * Personalise Cadmium with it's many configuration options to make it work the way you want.

## Download instructions (Recommended)
This section will help instruct you through the download process, please note that this is for the prebuilt executable file. If you wish to build or run the source code please follow this [tutorial](./tutorials/building_cadmium.md) instead. 

1. Download the latest version of Cadmium for your system from [*here*](https://github.com/Jodenee/Cadmium/releases).
2. Once downloaded unzip and extract the folder.
3. Open the "Cadmium" folder and run Cadmium.exe file.
4. You're all set! Happy downloading!
